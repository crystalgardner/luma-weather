## Overview
Luma Weather brings warmth and cuteness to your daily forecast. A soft, friendly alternative to robotic weather apps, Luma (meaning "light") is designed to brighten your day with sunshine and good vibes.

## Features
- **Smart Geolocation**: Finds you instantly.
- **Night Shift**: Colors adapt to the time of day.
- **Outfit Tips**: Tells you what to wear (and if you need an umbrella).
- **Privacy First**: Location data stays on your device.

## Tech Stack
Built with **React**, **Vite**, **TypeScript**, **Tailwind CSS**, and **Framer Motion**.

## Getting Started

```bash
# 1. Clone the repo
git clone https://github.com/crystalgardner/luma-weather.git

# 2. Install dependencies
npm install

# 3. Start development server
npm run dev
```
